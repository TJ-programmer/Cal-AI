## 🚀 Getting Started

### Prerequisites
- Node.js (for client & server)
- Python 3.9+ (for backend services)

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/TJ-programmer/Cal-AI.git
   cd Cal-AI
   ```

2. **Set up the client**
   ```bash
   cd client
   npm install
   npm start
   ```

3. **Set up the server**
   ```bash
   cd ../server
   npm install  # or pip install -r requirements.txt if using Python
   npm start    # or python app.py
   ```

## 💡 How It Works

Cal-AI combines a sleek frontend interface with intelligent backend processing to deliver a next-generation calendar experience. The AI engine analyzes your schedule patterns to suggest optimal times for meetings and tasks.
